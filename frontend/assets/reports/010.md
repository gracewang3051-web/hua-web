## 第二部分：练习题参考答案

1. **什么是议论文写作中的“母题”？**
   母题是作文题目中反复出现的、具有通用性的核心话题（如“新与旧”、“经验”等）。它像是一个内容和逻辑的库，掌握母题后可以举一反三，将通用的理论、例子与具体题目关键词相捆绑，实现标准化的审题和应用。

2. **简述“咖啡豆浆模型”（否定之否定的简化版）的核心操作流程。**
   该模型要求先分析自己最终不选的那个事物（如豆浆）的优缺点，展示对其价值的深度思考；随后通过引入特定的前提条件（如“需要提神”），自然地转入对最终选择（如咖啡）的论证。这种方式能使逻辑链条显得完整、深入，避免片面化。

3. **在论证中，为什么要先分析自己“不选”的那个选项（如豆浆）？**
   先写不选的选项可以展示论证者经过了深入且有逻辑线条的思考，即便不选也能看到其优点。这种处理方式能够增强结论的安全感和说服力，证明最终的选择是权衡正反两面后得出的理性结论。

4. **根据课程内容，新生事物（新）通常具备哪些优越性？**
   新生事物能满足人们的新需求，适应社会发展，并弥补旧事物的不足。它们通常具有更高、更完善、更先进的特点，具备更强的适应能力，能冲击美好未来的心理预期，体现出高效、便捷和先进性。

5. **为何不能简单地认为“新总比旧好”？请从现代性角度简述。**
   新事物由于缺乏时间沉淀，可能存在不成熟、不完善的隐患。在现代性背景下，过度依赖新科技（如AI、导航）可能导致人的自主控制权和决策权被“掠夺”，甚至让个体陷入“现代性的牢笼”或“996暴政”，忽略了个体的丰富性与审美。

6. **“旧”的事物在社会发展中具有哪些不可替代的价值？**
   旧事物经过时间淬炼，具有深厚的厚重性、完整性和普世性。它们不仅是一个城市或个体的“来时路”，提供了人文吸引力和身份认同，还能在与新事物的交互中提供养分，作为一种底蕴防止社会陷入虚无主义。

7. **新与旧之间的“对立统一”关系是如何体现的？**
   对立体现为新事物对旧事物的否定与冲击，往往伴随着利益的损益和观念的斗争。统一则体现为否定并非全盘抛弃，而是“克服与保留”的结合，即新事物吸取旧事物的精华，旧事物在新的形态中焕发生机。

8. **文中提到的王安石与司马光之争，其实质是什么？**
   实质是革新派与保守派的政治与逻辑斗争。王安石代表追求打破旧枷锁的新生力量，而司马光则代表重视传统秩序、强调通过既有经验保障安全的保守力量，反映了新旧事物在更替过程中的残酷斗争与复杂心态。

9. **以上海新天地的改造为例，说明新旧事物如何实现“交互关系”。**
   新天地将原有的民居（旧）改变为商业或酒店等新功能，使其在保留历史风貌的同时焕发新的生命力。这种模式证明了新事物可以在旧事物中寻求滋养，而旧事物通过新的改造获得动力，实现两者的相互提升。

10. **建立高中作文“标准模型”应从哪两个维度入手？**
    一是“内容线”，即归纳话题及其相关的例子、理论库；二是“逻辑线”，即明确命题模型、逻辑关系（如A与B的关系）以及论证的结构导引。

## 第三部分：论文/论述题建议

1. **论“现代性异化”：** 结合课程中关于导航系统、AI（如豆包）对人类决策权的影响，探讨在追求“新”与“高效”的过程中，人类如何避免迷失在工具理性的“牢笼”中。
2. **新旧平衡的城市哲学：** 以西安古迹保护与上海浦东开发为例，论述在现代化进程中，如何实现“新城”与“老城”的和谐共生，并阐述这种共生的社会文化价值。
3. **论苏轼的“舆论影响力”与“政治站位”：** 根据资料中对苏轼作为“大V博主”人设的解读，分析知识分子在面对新旧交替（如王安石变法）时，其审美追求与现实政治逻辑之间的冲突。
4. **否定之否定的思辨艺术：** 深入探讨为什么“论点自由”建立在“论据库完整”的基础上。如何在作文中通过博弈模型（如咖啡豆浆模型）提升议论文的思维层次？
5. **传统事物的“历久弥新”：** 讨论在实体店受网购冲击、实体菜场受外卖挑战的今天，具有“人文厚度”的旧事物应如何通过“推陈出新”证明其不可替代的价值。

## 第四部分：核心术语表

| 术语 | 定义与解释 |
| :--- | :--- |
| **母题 (Mother Topic)** | 议论文中具有普世性和反复性的核心议题（如新旧、经验、得失等），是建立作文体系的基础。 |
| **内容线与逻辑线** | 审题的两大支柱。内容线关注话题涉及的资源与素材；逻辑线关注命题的结构关系与论证推导。 |
| **去倾向化** | 上海等地区高考阅卷的一种趋势，不预设论点的对错倾向，主要考查文章的逻辑自洽性和思维深度。 |
| **咖啡豆浆模型** | 即“否定之否定简化版”，一种先分析次选方案优缺点，再引出首选方案，以增强论证严密性的结构。 |
| **现代性 (Modernity)** | 指向高速发展、理性化、标准化的现代社会特征，文中常讨论其对个体生存空间的异化作用。 |
| **异化 (Alienation)** | 在发展中事物走向其反面。如人类发明工具是为了更自由，最后却被工具（如算法、996）所束缚。 |
| **去媚 (Disenchantment)** | 文中引自尼采的概念，指打破旧有的神秘或迷信认知，代之以理性的、科学的（新的）认知体系。 |
| **对立统一** | 辩证法核心。新事物否定旧事物形成对立，同时又继承其精华、相互依存，形成统一。 |
| **交互关系** | 新旧事物并非单向取代，而是相互滋养：新事物提供动力，旧事物提供底蕴和养分。 |
| **虚无主义 (Nihilism)** | 指盲目追求“新”而全盘否定“旧”的传统和历史，导致精神家园丧失、价值评判缺失的倾向。 |
